# Expense Tracker (React)

A simple and responsive Expense Tracker application built using **React**.  
Users can add and delete expenses with date and amount.

---

##  Live Demo
 https://expense-tracker-pink-tau.vercel.app/

---

##  Tech Stack
- React (Vite)
- JavaScript (ES6)
- CSS (Flexbox)
- Git & GitHub
- Vercel (Deployment)

---

##  Features
- Add new expenses
- Delete expenses
- Date-wise expense display
- Clean and centered UI
- Fully deployed on Vercel
